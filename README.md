# Photo Audit Workspace (摄影平台供稿审核后台 MVP)

## 📖 项目简介
本项目是一个专为摄影平台设计的供稿审核后台纯前端交互 Demo。在 UGC（用户生成内容）海量增长的背景下，平台面临着极高的内容合规风险（如版权争议、政治敏感、色情低俗）以及画质把控难题。
传统的人工审核模式效率低下且极易漏判。本项目旨在通过**“AI 初筛 + 人工复核”**的人机协同模式，解决**审核人力成本高、风险内容漏放率高、创作者等待时间长**三大核心业务问题，提供一个高效、流畅、专业的审核工作台。

## 🎯 核心业务指标 (Core Metrics)
本系统在设计与迭代中，重点监控以下三大核心指标：
- **机审召回率 (Recall)**：AI 成功拦截的违规/低质图片占实际所有违规/低质图片的比例。本项目设定目标为 **99.2%**，以确保极低的内容安全“漏放”风险，守住平台合规底线。
- **机审准确率 (Precision)**：AI 拦截并打回的图片中，真正属于违规/低质的比例。高准确率能有效减少对优质稿件的“误伤”，降低人工复核的无效工作量，保障摄影师体验。
- **平均出审时效 (Average Review Time)**：单张图片从进入队列到审核完成的平均耗时。通过前端沉浸式 UI、便捷的交互和 AI 预判，目标将人工决策时间压缩至 **1.2秒/张**。

## 🤖 AI 技术栈应用 (AI Tech Stack)
虽然本 Demo 为前端静态交互演示，但在规划的实际后端架构中，AI 赋能是整个审核链路的核心引擎。系统设计的底层 AI 技术栈包括：
- **视觉-语言大模型 (VLM, 如 CLIP/BLIP)**：用于深度理解图像语义，进行全网版权溯源比对，以及复杂的敏感元素（如特定标志、受管控物品）识别。
- **图像美学质量评估模型 (IQA, 基于 ResNet 等架构)**：用于评估“画面模糊”、“曝光异常”、“噪点过多”等构图与画质问题，拦截低质量废片。
- **OCR (光学字符识别)**：精准提取图片中的文本，检测是否包含违规文字、恶意引流广告或敏感标语。
- **NSFW 专项分类器**：基于深度卷积神经网络专项训练，用于涉黄、暴力、血腥等绝对红线内容的极速拦截。

## ✨ 核心功能亮点
- **沉浸式暗黑工作台**：专为长时间审图设计的护眼 UI，降低审核员视觉疲劳。
- **AI 智能标签辅助**：每张图片附带 AI 机审建议（如“建议通过”、“疑似版权风险”、“画面模糊”），辅助人工快速做出决策。
- **精细化打回与申诉闭环**：支持细分打回原因（版权争议、画面模糊、政治敏感），并自带摄影师申诉入口与联动日志。
- **实时业务看板与日志**：动态展示今日已审数量、出审时效及召回率，操作日志实时留存。
- **状态持久化**：使用 `localStorage` 模拟后端数据持久化，刷新页面进度不丢失。
